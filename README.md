# Ovena Health Inventory Portal

A polished inventory management portal for tracking Ovena Health stock across Amazon and Shopify with a more intuitive, dashboard-style experience.

## Included inventory catalog

- Collagen Wound Dressing: 2"x2", 4"x4", 7"x7"
- Gauze Rolls
- Silicone Foam Dressing: 4"x4", 6"x6", 8"x8"
- Disposable Gloves
- Wound Wash
- Compression Socks
  - Versions: knee high closed toe, knee high open toe, thigh high closed toe, thigh high open toe
  - Sizes: 1 through 5
  - Colors: black and beige

## What makes this “ultimate” UI version

- Split layout with a focused dashboard workspace and a left-side sock version guide.
- Improved visual hierarchy with KPI cards, modern "glass" panel styling, gradients, and clearer call-to-action buttons.
- Cleaner table structure focused on primary operations (SKU, product, variant, channel quantities, reorder, status).
- Maintains all core workflows from prior versions: inline editing, local persistence, filtering, and CSV export.

## Features

- Editable Amazon and Shopify quantity columns
- Computed total inventory and low-stock status per SKU
- Reorder-level editing
- Search and channel filtering
- CSV export for filtered rows
- Local persistence in browser `localStorage`

## Run locally

Open `index.html` in your browser.
